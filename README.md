# washing-machine-controller

**Apache License 2.0** に基づく、多言語（C / C# / Java）で実装された洗濯機制御のシミュレーション・プロジェクトです。

## プロジェクト概要
このリポジトリは、洗濯機の基本的な動作ロジック（給水・洗い・すすぎ・脱水）と状態管理を、異なるプログラミング言語で実装した学習用・プロトタイプ用コード集です。

それぞれの言語特性を活かした設計を行っています：
- **C言語**: 手続き型による、ハードウェア制御に近いシーケンシャルな工程処理。
- **C#**: プロパティとフラグ管理による、堅牢なステート（状態）制御。
- **Java**: オブジェクト指向の標準的な作法に基づいたクラス設計。
